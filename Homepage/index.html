Cyber Security Training

Using cloud computing services, we have designed an interactive web based security training application. Our application contains sixteen online and on-demand training exercises. The purpose of these exercises is to ingrain and scale knowledge of security attacks, as well as, helping to mitigate concerns during deployment. This application was developed using AWS (Amazon Web Services), which is a secure cloud services platform. AWS allows their customers to “build sophisticated applications with increased flexibility, scalability and reliability.” AWS also offers their customers increased computer power, computing resources, content delivery and other functionalities.

The application design includes a homepage with links to all lessons. Each lesson contains an overview, a discovery phase, remediation phase, a prevention phase, and a  PDF summary. The lesson overview comprises of a realistic scenario of the attack. The discovery explains what to look for to see if anything has been compromised. The remediation and prevention phases review how to protect yourself from such an attack in the present and future. Finally the PDF incorporates a summary of the scenario and the the phases. This allows for easy review or it allows for the information to be accessible without going through the steps.
